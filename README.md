# 🚀 Vibe - 바이브 웹/앱 개발 스터디 프로젝트

> AI와 함께하는 협업 개발 경험, 코딩 없이 만드는 실전 웹 애플리케이션

<br/>

## 📖 프로젝트 소개

**Vibe**는 바이브 코딩(AI 기반 개발)을 통해 실전 소셜 웹 애플리케이션을 만들어가는 **오픈소스 스터디 프로젝트**입니다.

직접 코드를 타이핑하는 대신, AI와 대화하며 개발하는 새로운 방식으로 웹/앱 개발을 경험하고,
Git을 활용한 협업 방식을 배우며, 실제 배포까지 경험할 수 있습니다.

### 🌟 프로젝트 특징

- **💬 소스 코딩 금지** - 코드는 AI에게 맡기고, 당신은 설계와 기획에 집중하세요
- **🤝 실전 협업 경험** - Git PR을 통한 협업, 코드 리뷰 경험
- **🌐 직접 배포** - 본인의 Firebase/Supabase를 연결하고 직접 사이트 배포
- **🎯 실용적인 기능 구현** - 인증, 게시판, 채팅, 알림 등 실제 서비스에 필요한 기능들
- **📚 함께 성장** - 모르는 것을 서로 물어보고 배우는 학습 문화

<br/>

## 🎯 이런 분들께 추천합니다

- ✅ AI 기반 개발(바이브 코딩)에 관심이 있으신 분
- ✅ Git/GitHub를 실전에서 사용해보고 싶으신 분
- ✅ Firebase, Supabase 같은 백엔드 서비스를 배우고 싶으신 분
- ✅ Next.js, React 기반 모던 웹 개발을 경험하고 싶으신 분
- ✅ 혼자가 아닌 함께 성장하는 스터디를 원하시는 분

<br/>

## 🚀 시작하기

### 참여 방법

1. **이 저장소를 Fork** 하세요
2. **본인의 환경 설정**을 추가하세요
   - Firebase 프로젝트 생성 및 `.env.local` 설정
   - Supabase 프로젝트 생성 (선택)
3. **기능 개발 및 PR 제출**
   - 새로운 기능을 추가하거나 버그를 수정하세요
   - Pull Request를 보내주세요
4. **본인의 사이트에 배포**해보세요

### 개발 환경 셋업

```bash
# 저장소 클론
git clone https://github.com/your-username/vibe.git
cd vibe

# 의존성 설치
npm install

# 환경 변수 설정
cp .env.example .env.local
# .env.local 파일에 본인의 Firebase 설정 추가

# 개발 서버 실행
npm run dev
```

## 🛠️ 기술 스택

### 프론트엔드
- **Next.js 16** - React 기반 풀스택 프레임워크
- **React 19** - 최신 React 기능
- **TypeScript** - 타입 안전성
- **Tailwind CSS 4** - 유틸리티 기반 스타일링
- **shadcn/ui** - 접근성 높은 UI 컴포넌트

### 백엔드 & 데이터베이스
- **Firebase** - 인증, Realtime Database, Firestore
- **Supabase** (선택) - PostgreSQL 기반 백엔드

### 개발 도구
- **Turbopack** - 빠른 번들링
- **Claude Code** - AI 기반 개발 도구

<br/>

## 📚 스터디 진행 방식

### 세미나 시간 집중 개발
- **세미나 시간에만 개발/스터디 진행** - 혼자 공부하는 부담 없음
- **실시간 질의응답** - 막히는 부분은 바로 물어보고 해결
- **협업 경험** - Git PR, 코드 리뷰 등 실전 협업 방식 학습

### 바이브 코딩 원칙
- **코드 직접 타이핑 금지** - AI에게 요청하여 구현
- **설계와 기획에 집중** - "무엇을" 만들지, "어떻게" 동작할지 고민
- **AI와 대화** - 명확한 요구사항 전달 능력 향상

### 서로 돕는 문화
- GitHub 사용법 (Fork, Clone, PR 방법)
- Firebase, Supabase 셋업 방법
- 프론트엔드/백엔드 개발 질문
- 배포 및 도메인 연결

<br/>

## 📂 프로젝트 구조



```
vibe/
├── app/                    # Next.js App Router 페이지
│   ├── auth/               # 인증 관련 페이지
│   │   ├── signup/         # 회원가입
│   │   └── login/          # 로그인
│   ├── user/               # 사용자 관련 페이지
│   │   └── profile/        # 프로필 관리
│   ├── chat/               # 채팅 관련 페이지
│   │   └── room/           # 채팅방
│   ├── layout.tsx          # 루트 레이아웃
│   └── page.tsx            # 홈페이지
├── lib/                    # 유틸리티 및 Firebase 설정
│   ├── firebase.ts         # Firebase 초기화
│   ├── auth.ts             # 인증 함수
│   ├── user.ts             # 사용자 관리 함수
│   └── chat.ts             # 채팅 함수
├── components/             # React 컴포넌트
├── docs/                   # 프로젝트 문서
│   ├── user.md             # 사용자 관리 시스템 문서
│   └── chat.md             # 채팅 시스템 문서
└── .env.local              # 환경 변수 (각자 설정)
```
